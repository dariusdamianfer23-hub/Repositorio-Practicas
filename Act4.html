Actividad 4: Instalación y configuración de dispositivos de RED.

Curso 2025/26 - Prácticas.

En esta actividad realizaremos la configuración completa de un switch y un router, creando VLANs, configurando direcciones IP y habilitando servicios de red.

Configuración del Switch

Para acceder a la interfaz gráfica del switch lo primero será cambiar la IP para que podamos conectarnos correctamente.

Desde cualquier navegador ponemos la puerta de enlace en el buscador.

Se abrirá una página donde deberemos introducir:

Usuario: admin

Contraseña: Campusfp1234

Configuración de VLAN

En la pestaña L2 FEATURES iremos a VLAN y pulsaremos Add.

Asignaremos:

ID: 70

Nombre VLAN: DARIUS

Seleccionaremos el puerto conectado al router y los puertos usados para servidor y PC.

Finalmente pulsamos Create.

Después vamos a Port Config y asignamos los puertos a la VLAN creada.

Configuración del Router

Primero creamos las interfaces VLAN desde:

Interface > VLAN > botón +

Asignamos:

- Nombre de la VLAN

- ID de VLAN

- Interfaz: bridgeLocal

Configuración IP y DHCP

Para asignar direcciones IP vamos a:

IP > Addresses > botón +

Añadimos la dirección IP correspondiente a cada interfaz.

Después configuramos DHCP desde:

IP > DHCP Server > DHCP Setup

Configuramos:

- Dirección IP y máscara

- Puerta de enlace

- DNS de Google

Configuración del Bridge

Para configurar el bridge vamos a:

Bridge > VLANs > botón +

Agregamos:

- El Bridge principal

- El puerto físico ether3

Guardamos cambios y verificamos conectividad.
