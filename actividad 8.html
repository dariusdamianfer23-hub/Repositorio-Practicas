Breve descripción

En esta actividad describiremos el proceso de configuración de un servidor NAS con OpenMediaVault y cómo subir archivos al mismo. Primero se accede a la interfaz web del NAS mediante su IP, usando las credenciales de administrador. Una vez dentro, se habilita el servicio SMB/CIFS y se configura el grupo de trabajo. Luego se crea una carpeta compartida asignándole un nombre, se crea un usuario y se configuran los permisos para que dicha carpeta sea accesible en red.

Para subir documentos al NAS existen dos métodos: el primero consiste en acceder desde el explorador de archivos de Windows usando la IP del NAS y las credenciales del usuario creado, arrastrando y soltando los archivos en la carpeta compartida. El segundo método es por línea de comandos, utilizando el comando scp desde PowerShell para transferir archivos directamente a una ruta absoluta del servidor.
